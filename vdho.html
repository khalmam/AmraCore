8. Troubleshooting

Issue Possible Cause Solution
Authentication Unsuccessful Wrong password or TLS mismatch Regenerate app password; verify TLS/port
Email Delay Worker not running Run: bench start or enable background workers
Sends from Admin Email Default outgoing not set Enable Default Outgoing on noreply@vhdo.org
Email Not Received Spam filters or DNS issues Check spam folder; verify SPF/DKIM records
Wrong Sender in Templates Account not linked Set correct Email Account in Print Format or Email Template
